Exemplos de utilização da biblioteca Matplotlib com 4 tipos de gráfico(Gráfico de linha, barra, pizza e dispersão)

Para rodar o código terá que instalar as extensões Matplotlib para Python através do comando "conda install -c conda-forge matplotlib" que deve ser executado no cmd. Em seguida baixe a pasta ZIP com todos os arquivos, extraia e depois apenas abra o arquivo graficos.py e execute-o.

Há duas formas de baixar o arquivo ZIP. A primeira é apenas clicando ao lado direito da tela em tags, depois em python, depois disso irá aparecer o arquivo Source code.zip, basta clicar em cima para baixar

O outro método para baixar a pasta como arquivo ZIP é clicando no ícone verde no canto superior direito "<> Code" e clique em "Download ZIP", mas nesse método provavelmente terá que extrair duas vezes.
